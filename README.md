Random Password Generator

Автор

Вафина Айгуль

---

Описание

Данное приложение представляет собой графический генератор случайных паролей.
Пользователь может настраивать параметры генерации, включая длину пароля и типы используемых символов.

Программа разработана с использованием Python и библиотеки Tkinter.
Поддерживается сохранение истории сгенерированных паролей в JSON-файл и их загрузка при повторном запуске приложения.

---

Функциональные возможности

- Генерация случайных паролей
- Настройка длины пароля (ползунок)
- Выбор типов символов:
  - цифры
  - буквы
  - специальные символы
- Валидация пользовательского ввода
- Отображение истории сгенерированных паролей
- Сохранение истории в JSON
- Загрузка истории при запуске программы

---

Установка и запуск

1. Убедитесь, что установлен Python 3
2. Скачайте проект или клонируйте репозиторий:

git clone <ссылка_на_репозиторий>

3. Перейдите в папку проекта:

cd password-generator

4. Запустите приложение:

python main.py

---

Примеры использования

1. Установите длину пароля с помощью ползунка
2. Выберите нужные типы символов
3. Нажмите кнопку "Сгенерировать"
4. Полученный пароль появится в поле ввода
5. Все сгенерированные пароли сохраняются в таблице истории

Пример сгенерированного пароля:

A7f#K2m!q

Особенности реализации

- Используется модуль random для генерации паролей
- История хранится в формате JSON
- Реализована проверка корректности ввода:
  - минимальная и максимальная длина
  - выбор хотя бы одного типа символов
- Интерфейс реализован с использованием Tkinter
